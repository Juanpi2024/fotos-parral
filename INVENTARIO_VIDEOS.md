## Para transferir el proyecto

### Opción A: Transferir canal de YouTube

1. Crear canal dedicado: "Historia de Parral" o similar
2. Subir todos los videos al nuevo canal
3. Actualizar los IDs en el código
4. Transferir propiedad del canal al comprador

### Opción B: Incluir videos como archivos

1. Descargar cada video en formato MP4
2. Crear carpeta `videos/` en el proyecto
3. Reemplazar iframes por etiquetas `<video>`
4. Entregar proyecto con todos los archivos

### Búsqueda rápida para reemplazo

```bash
# Buscar todos los iframes de YouTube en el proyecto:
grep -r "youtube.com/embed" *.html
```
